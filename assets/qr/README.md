# Recursos del QR

`location-logo.jpg` es el recurso gráfico conservado junto al QR aportado al repositorio. Esta carpeta no se copia automáticamente a Cloudflare.

El archivo descargable que sí se publica está en:

```text
public/downloads/crevi-loc-web-qr.png
```

Si se regenera el QR, debe seguir apuntando a la URL pública de Crevi Loc Web y hay que sustituir el archivo de `public/downloads/` antes de ejecutar el build.
